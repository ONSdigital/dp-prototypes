/search/code-lists

POST

Get a list of codes containing information about dimensions which are used for all datasets at the ONS

Parameters

Name Type Description
q query Query all code lists
offset query The offset from the start of results. This defaults to 0
limit query The maximum number of results to be returned. This defaults to 20

Responses

200

A Json message containing a set of code lists

400

Missing parameters within request or invalid query

500

Failed to process the request due to an internal error
