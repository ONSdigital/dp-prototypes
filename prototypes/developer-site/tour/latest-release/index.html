Find the latest release

Once you have a dataset ID, you can find the latest release of a dataset using the get dataset endpoint.

A link to the latest release can be found in links.latest_version field.

Example

Make a GET request to https://api.beta.ons.gov.uk/v1/datasets/cpih01
