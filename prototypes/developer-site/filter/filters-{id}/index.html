/filters/{id}

GET

Get document describing the filter

Responses

200

The filter was found and document is returned

404

Filter not found

500

Failed to process the request due to an internal error

PUT

Update the filter by providing new properties, submit a filter for processing by setting query parameter `submitted` to `true`

Parameters

Name Type Description
submitted query A flag to indicate the submission of a filter
filter body Model of all editable properties within a filter
id path The unique filter ID for customising a dataset

Responses

200

The filter job has been updated

400

Invalid request body

404

Filter not found

422

Unprocessable entity - instance has been removed

500

Failed to process the request due to an internal error
